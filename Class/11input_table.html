form : user에게 data를 입력받는 문서 양식 -> 서버 전송

form action : data를 전송할 주소를 적는 곳.

form method : data를 전송하는 방법.

get : 주소창에 data가 보이는 전송 방식. 보안 취약. 속도 빠름. 256byte만 전송 보장

post : 주소창에 data가 안보이는 전송 방식. 보안 OK. 속도 느림. 대용량 전송 가능

input : data를 입력 받는 객체.

id 속성 : html page에서 객체(element,요소)를 구분하는 이름

name 속성 : 서버로 전송하는 data에 붙는 이름
